How to Command Dude

วิธีสั่งงานน้องดู๊ด — Your AI Personal Assistant

Register / ลงทะเบียน

สมัครสมาชิกเพื่อเข้าถึงระบบ AI และเครื่องมือทั้งหมดของ DuDe Hawaiian

Example Command

> ลงทะเบียน
> สมัคร
> register

IoT Control / ควบคุมอุปกรณ์

ควบคุมไฟและอุปกรณ์ Smart Home ได้จากทุกที่ผ่าน LINE

Example Command

> เปิดไฟหน้าบ้าน
> ปิดไฟห้องนอน
> สถานะไฟ

Search / ค้นหา

ค้นหาข่าวสาร ราคาสินค้า หรือสภาพอากาศจากอินเทอร์เน็ตแบบเรียลไทม์

Example Command

> ค้นหา ราคาทองวันนี้
> search ข่าววันนี้
> ค้นหา พยากรณ์อากาศ

Knowledge / ค้นในความรู้

ถามคำถามจากฐานข้อมูลเอกสารหรือคู่มือที่เคยอัปโหลดไว้ (RAG)

Example Command

> rag วิธีประกอบคอม
> ค้นในความรู้ ขั้นตอนเบิกจ่าย
> สรุปไฟล์ที่ส่งไปล่าสุด

Caclulator / เครื่องคิดเลข

คำนวณสูตรคณิตศาสตร์หรือแปลงหน่วยอย่างรวดเร็ว

Example Command

> 1250 * 7%
> 100 ดอลลาร์ เป็นกี่บาท
> 500 + 1500 / 2

Notes / บันทึก/เตือน

สั่งให้ช่วยจำโน้ตสำคัญ หรือตั้งการแจ้งเตือนตามวันและเวลาที่ต้องการ

Example Command

> บันทึกว่า จ่ายค่าเช่าแล้ว
> เตือนให้โทรหาแม่ ตอน 6 โมงเย็น
> note: ของที่ต้องซื้อ

Booking / จองบริการ

จองห้องพัก ร้านอาหาร หรือนัดหมายผ่านระบบ QloApps

Example Command

> หาห้องพัก
> จองโต๊ะ ร้านริมน้ำ
> ดูรายการจองของฉัน

Health / บันทึกสุขภาพ

บันทึกข้อมูลสุขภาพ น้ำหนัก ความดัน หรือประวัติการแพ้ยา

Example Command

> บันทึกน้ำหนัก 70 กก.
> ดูสมุดสุขภาพ
> แจ้งแพ้ยา แอสไพริน
Pro Tips

Elite Usage / เคล็ดลับ

ใช้งานน้องดู๊ดให้เต็มประสิทธิภาพด้วยเทคนิคต่างๆ

  • ▪ พิมพ์ได้ทั้งไทยและอังกฤษอย่างเป็นธรรมชาติ
  • ▪ ใช้ "ค้นหา" สำหรับข้อมูลจากอินเทอร์เน็ต
  • ▪ ใช้ "rag" สำหรับข้อมูลในไฟล์เอกสารของคุณ
